# WT3: Real-time Weather App

This project is a simple and elegant Real-time Weather App built using HTML, CSS, and JavaScript. It fetches live weather data from the OpenWeatherMap API and displays it with a clean, modern UI.The app allows users to enter any city name and instantly view real-time temperature, weather conditions, humidity, and wind speed — all beautifully presented inside a responsive weather card.

## Features
- Real-time Weather Data — Uses the OpenWeatherMap API to fetch live updates.
- Auto Refresh — Updates automatically every 10 seconds.
- Modern UI — Gradient background with glass-like weather cards.
- City Search — Instantly check weather for any location worldwide.
- Responsive Design — Works smoothly on desktop and mobile.

## Technologies Used

- HTML5 — Page structure and layout
- CSS3 — Styling with gradients, shadows, and hover effects
- JavaScript (ES6) — API integration and dynamic content rendering
- OpenWeatherMap API — Real-time weather data source

## How It Works

- Enter a city name in the input field.
- The app sends a request to the OpenWeatherMap API.
- The weather data is displayed in a visually appealing card format.
- The weather information auto-refreshes every 10 seconds.

# WT4: User Registration Form

This project is a simple and modern user registration web page built using HTML, CSS, and JavaScript.
It validates user input fields such as name, email, password, and mobile number, returning JSON validation messages or a success message when all fields are valid.

## Features

- Input validation for name, email, password, and mobile number
- Instant feedback using JSON-style error messages
- Clean, minimal, and professional UI
- Fully responsive and easy to use

## Technologies Used

- HTML5 — Structure of the form
- CSS3 — Styling with shadows, transitions, and layout
- JavaScript (ES6) — Input validation and dynamic output
