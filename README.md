# WT1: SmartGadget 3000 Landing Page

This repository contains the source code for the SmartGadget 3000 product landing page. It features a modern, responsive design with sections for:

- Hero banner with call-to-action
- Product features overview
- Customer testimonials
- Pricing plans

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Responsive design principles
