# WT2: Clock + Code Comparison

This repository contains the source code for a web page that combines a real-time digital clock and a side-by-side comparison of React and Angular code snippets. It highlights differences between these popular frontend technologies with a clean and modern UI.

## Features

- Real-time clock with animated effects on minute change
- Side-by-side React and Angular code examples
- Responsive comparison table showing key distinctions
- Modern CSS styling with gradients and animations

## Technologies Used

- HTML5
- CSS3 with Flexbox and Grid
- JavaScript
